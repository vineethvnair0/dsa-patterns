Problem: Implement Trie (Prefix Tree)

Implement a trie with insert, search, and startsWith methods. A trie is a tree-like data structure used to store and retrieve strings efficiently.

Example:
Operations: insert("apple"), search("apple"), startsWith("app")
Output: true, true (found "apple", prefix "app" exists)

How Trie works:

  • Trie (Prefix Tree) stores strings character by character in a tree structure
  • Each node contains: array of children (26 for lowercase) and isEndOfWord flag
  • Insert: Traverse/create nodes for each character, mark last as end
  • Search: Traverse nodes for each character, check if last node is marked as end
  • StartsWith: Traverse nodes for prefix, return true if all characters found
  • Time complexity: O(m) for all operations where m = word length
  • Space complexity: O(ALPHABET_SIZE * m * n) where n = number of words
