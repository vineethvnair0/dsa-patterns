Problem: Reverse Linked List

Given the head of a singly linked list, reverse the list in-place and return the new head.

Example:
Input: head = [1,2,3,4,5]
Output: [5,4,3,2,1]

How it works:

  • Use three pointers: prev (null), current (head), next (null)
  • Iterate through the list:
  • 1. Save next node: next = current.next
  • 2. Reverse link: current.next = prev
  • 3. Move prev forward: prev = current
  • 4. Move current forward: current = next
  • After iteration, prev will be the new head
  • Time complexity: O(n), Space complexity: O(1)
